# Aperfeiçoamento em HTML e CSS com Bootstrap

O projeto foi desenvolvido com o objetivo de aprofundar conhecimentos técnicos em desenvolvimento front-end, focando no framework Bootstrap.

## Tecnologias Utilizadas
* **HTML**: Estruturação semântica.
* **CSS**: Estilizações personalizadas e overrides.
* **Bootstrap**: Sistema de grid, componentes e utilitários.
  
## Conceitos Praticados
* **Sistema de Grid (Flexbox):** Organização de layouts complexos usando `container`, `row` e `col`.
* **Componentes:** Implementação de Navbar, Cards, Modais e Botões.
